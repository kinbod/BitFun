# 粉色波普（Pink Pop）

## 风格标识
`pink-pop`

## 适用场景
人物介绍、产品亮点、生活方式、年轻化品牌、社群内容、创意提案等需要亲和力与潮流感的演示。

## 风格定位
Pink Pop。以哑光粉底为识别色，提供两种力度：精致编辑杂志（成熟可爱）与街头波普（活泼能量）。根据主题选定一种力度后全 deck 保持一致。

## 视觉系统
- 背景使用哑光柔和粉色（#fce7f3）。
- 精致编辑力度：中央大幅主体区域（大圆角矩形/有机形状），边缘非对称放置对话气泡、竖排文字框和手写批注；四角可加细线裁切标记，营造杂志打样感。
- 街头波普力度：粗线条（border: 3px solid）、轻微夸张变形的几何形状、Q 弹有机色块、涂鸦线条、手绘箭头和贴纸标签。
- 两种力度都要求信息层级清楚，装饰不得淹没正文。

## 配色（CSS 可用）
- 背景：#fce7f3（柔和粉）
- 文字：#831843（深粉紫红）或 #374151（炭灰，精致力度）
- 强调：#db2777 / #be185d（亮粉/深粉）
- 点缀：#f472b6（浅亮粉）
- 面板：#fdf2f8（浅粉）

## 排版（HTML/CSS）
- 精致力度：标题用高对比衬线体（Georgia, "Songti SC", serif; font-weight: 700）；批注可用手写体。
- 街头力度：标题用粗体无衬线（font-weight: 900; font-size: 28-40px）；文字可略微倾斜错位（rotate(-2deg)）但保证可读。
- 内容拆成 NO.1/NO.2 编号栏目、短句、标签或贴纸卡片；页面文字使用用户指定语言，少量英文作栏目标签。

## 推荐版式（960×540pt）
- 主体区域居中，编号信息分布两侧（flex 布局）。
- 一侧竖排标题（writing-mode: vertical-rl），另一侧气泡要点。
- 有机形状拼贴 + 手绘批注；贴纸式卡片沿对角线排列（rotate(-5deg)）。
- 观点页用巨大关键词和粗线箭头表达方向。

## CSS 实现要点
- 有机形状：`border-radius: 60% 40% 30% 70% / 60% 30% 70% 40%;`
- 气泡：`border-radius: 20px 20px 20px 0` + 柔和阴影 `box-shadow: 2px 2px 8px rgba(0,0,0,0.05)`。
- 贴纸：`background: #fdf2f8; border: 2px solid #db2777; transform: rotate(-3deg); box-shadow: 3px 3px 0 #db2777;`
- 裁切标记：`position: absolute` 的细线 div（宽 20px，高 1px）。

## 避免
- 不使用过于幼稚的鲜艳撞色、夸张渐变或重度阴影。
- 不使用写实商务照片、玻璃拟态或低对比粉白文字。
- 不把所有内容居中堆叠；保留编辑版式的非对称张力。

## 输出要求
保留输入内容的信息结构和文字含义，按此风格重绘为统一连贯的 16:9 演示页面。所有视觉元素必须用 HTML/CSS 实现，不依赖外部图片资源。
