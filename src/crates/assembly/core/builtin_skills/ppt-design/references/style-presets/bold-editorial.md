# 黑白红大字（Black-White-Red）

## 风格标识
`bold-editorial`

## 适用场景
观点表达、行业分析、趋势报告、品牌封面、人物访谈、结论页等需要强文字冲击力与编辑设计感的演示。

## 风格定位
Bold Editorial。融合黑白红时尚编辑、现代报纸头版与杂志排版：白底黑色大字、红色点缀、非对称网格，用排版本身制造智性冲击。

## 视觉系统
- 背景以白色为主（#ffffff，可用冷灰白 #f5f5f0 变体），保留充足留白。
- 高品质排版和文字层级是视觉主体，而非图片；每页只表达一个核心信息。
- 版式严禁简单居中：优先左上、左下或边缘锚定的非对称构图；文字可大裁切、局部溢出（负边距或超大字号）。
- 红色只用于关键点缀：短线（width: 40px; height: 3px）、圆点、编号、关键词或强调框。
- 可用荧光笔式黄色高亮（background: linear-gradient(transparent 60%, #facc15 60%)）点缀关键数字或结论。

## 配色（CSS 可用）
- 背景：#ffffff（白色）
- 文字：#000000（黑色）
- 强调：#dc2626（红色）；高亮 #facc15（荧光黄，少量）
- 辅助：#525252（中灰）
- 面板：#fafafa（极浅灰）

## 排版（HTML/CSS）
- 标题用超粗无衬线（font-weight: 800-900; font-size: 36-56px; letter-spacing: -0.02em），占页面 30%-50%。
- 标题与正文字号比例极端（如 48px vs 14px），形成报纸头版式跳跃；副标题极小（12-14px）。
- 允许动态排版：竖排（writing-mode: vertical-rl）、跨层叠加、局部出血；红色文字只用于最关键的 1-2 个词。

## 推荐版式（960×540pt）
- 封面：2-5 个字或一个短词作视觉锚点 + 极小副标题，非对称布局。
- 结论页：一句金句占满画面，局部溢出边缘，红色短线作锚点。
- 数据页：巨大黑色数字（font-size: 80-120px）+ 红色单位/注释 + 高亮线。
- 观点页：左侧密集文字块与右侧大留白形成对比。

## CSS 实现要点
- 响应式大字：`font-size: clamp(24px, 5vw, 56px)`；溢出效果：`font-size: 120px; line-height: 0.9; overflow: hidden;`
- 红色点缀线：`width: 40px; height: 3px; background: #dc2626;`
- 非对称布局：`display: grid; grid-template-columns: 2fr 1fr;` 或绝对定位。
- 细线分隔：`border-top: 1px solid #1c1917; width: 60px;`

## 避免
- 不使用普通居中标题页、半吊子字号差异或平均分布布局。
- 不使用多色装饰、复杂渐变、重阴影或廉价素材；红色不大面积铺满。
- 不用密集文字削弱大字冲击力；不让视觉元素抢走标题主导权。

## 输出要求
保留输入内容的信息结构和文字含义，按此风格重绘为统一连贯的 16:9 演示页面。所有视觉元素必须用 HTML/CSS 实现，不依赖外部图片资源。
