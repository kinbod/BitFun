# 黄底黑字杂志（Yellow Magazine）

## 风格标识
`yellow-magazine`

## 适用场景
封面、品牌宣言、活动海报、时尚消费品、生活方式、个人 IP、趋势观点等需要高识别度的演示。

## 风格定位
Yellow Magazine。黄底黑字大排版加少量手写点缀，制造强烈杂志感与高级流行气质。

## 视觉系统
- 背景使用高识别度黄色（#facc15，或明亮不刺眼的 #fef08a 变体），作为整套页面的识别底色。
- 主视觉用大字排版、色块裁切或局部放大；标题、色块与贴纸元素可局部重叠，形成大胆构图。
- 加入少量手写字、箭头、圈注或散落贴纸，制造编辑现场感；只用于强调词或批注，不用于正文。

## 配色（CSS 可用）
- 背景：#facc15（明黄）
- 文字：#171717（黑色）
- 辅助：#ffffff（白色）和 #57534e（暖灰）
- 强调：#171717（黑色反白块）和 #dc2626（红色批注，少量）
- 面板：#fef08a（浅黄）

## 排版（HTML/CSS）
- 标题用大号高对比字体（font-weight: 800; font-size: 36-52px），可跨色块或溢出画面边缘。
- 正文用短句、编号或小标签承载，不使用长段落。
- 手写字体（font-family: cursive）仅用于批注和强调短语；字体总数控制在三类以内（标题/正文/手写）。

## 推荐版式（960×540pt）
- 巨大标题占半屏，色块从另一侧切入（grid: 1fr 1fr）。
- 中央大色块，四周分布短标签或手写批注。
- 单句宣言页：一个大词 + 一个小副标题。
- 列表页：贴纸编号分隔要点，像杂志栏目。

## CSS 实现要点
- 响应式大字：`font-size: clamp(32px, 6vw, 64px)`。
- 反白色块：`background: #171717; color: #facc15; padding: 20px;`
- 贴纸：`background: #ffffff; border: 2px solid #171717; transform: rotate(-5deg); box-shadow: 3px 3px 0 #171717;`
- 圈注：`border: 2px solid #dc2626; border-radius: 50%;`；箭头用 CSS 三角形或 → ↗ ↓。

## 避免
- 不使用复杂渐变、重阴影或廉价贴纸堆叠。
- 不让手写元素干扰核心信息阅读。
- 不让排版质量或裁切破坏高级感。

## 输出要求
保留输入内容的信息结构和文字含义，按此风格重绘为统一连贯的 16:9 演示页面。所有视觉元素必须用 HTML/CSS 实现，不依赖外部图片资源。
