# 黑白极简（Monochrome Minimal）

## 风格标识
`minimal-gallery`

## 适用场景
作品集、策略提案、咨询报告、摄影/设计展示、艺术项目、数据结论页等需要克制高级感的演示。

## 风格定位
Minimal Gallery。融合严格网格作品集与高端画册跨页：网格、留白、精确对齐、黑白灰层级，每一页都像画廊展墙或画册展开。

## 视觉系统
- 背景纯白或浅灰（#fafafa / #ffffff），必要时可用黑底强调页（#000000，黑底白字，用于强结论或章节切换）。
- 严格网格对齐所有元素；大面积负空间，主体通常占 50%-80%。
- 每页左上/右上角放小节编号与标题（如 01. INTRODUCTION）。
- 视觉主体可用低饱和色块、抽象几何或 CSS 形状；细线边框（1px solid #e5e7eb）作精致装帧。
- 可选「纸感方言」：暖灰白底（#f5f5f4）+ 折纸多边形（clip-path）+ 微妙折叠阴影，用于艺术/文化主题；选定后全 deck 保持一致。

## 配色（CSS 可用）
- 背景：#fafafa（浅灰）或 #ffffff（纯白）
- 文字：#171717（近黑）
- 强调：#171717（黑色粗线、深色巨大数字）
- 辅助：#737373 / #a3a3a3（灰阶）
- 面板：#ffffff（白色）；特殊页 #000000（黑底白字）

## 排版（HTML/CSS）
- 标题用现代无衬线（font-weight: 500-700; font-size: 24-40px; letter-spacing: 0.02-0.05em）。
- 正文小字号（12-16px），宽松字距与行高（line-height: 1.6）；页码极小（10px）。
- 层级靠字号、位置和留白制造，而不是多色装饰。

## 推荐版式（960×540pt）
- 标题页：分散式布局，中心小标题搭配角落编号和关键词。
- 数据页：左侧叙述，右侧超大黑色数字 + 细分隔线。
- 画册跨页：左右非对称，一侧大色块/图形，一侧留白。
- 网格展示：2×2 或 3×3 等距网格（gap 统一），文字极少。

## CSS 实现要点
- 网格：`display: grid; grid-template-columns: repeat(12, 1fr); gap: 20px;`
- 分隔线：`width: 40px; height: 1px; background: #171717;`
- 编号：`font-size: 12px; letter-spacing: 0.1em; color: #525252;`
- 装帧：`border: 1px solid #e5e7eb; padding: 8px;`；留白：`padding: 60px; gap: 40px;`
- 精致感：`letter-spacing: 0.05em; text-transform: uppercase;`

## 避免
- 不使用圆润可爱元素、渐变背景、复杂纹理或高饱和彩色块。
- 不使用随意对齐或装饰性过强的图标；不让文字过多破坏画册感。
- 极简风需要清晰层级和留白纪律。

## 输出要求
保留输入内容的信息结构和文字含义，按此风格重绘为统一连贯的 16:9 演示页面。所有视觉元素必须用 HTML/CSS 实现，不依赖外部图片资源。
