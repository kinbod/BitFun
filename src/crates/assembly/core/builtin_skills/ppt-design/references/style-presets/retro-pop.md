# 复古海报波普（Retro Poster Pop）

## 风格标识
`retro-pop`

## 适用场景
复古主题活动、音乐文化、品牌宣传、艺术展览、趋势话题等需要怀旧感和强记忆点的演示。

## 风格定位
Retro Pop。以复古色调、粗体排版和海报级冲击力为底盘，可选「古典雕塑混搭」方言制造超现实波普记忆点。选定方言后全 deck 保持一致。

## 视觉系统
- 背景使用复古色调（#fef3c7 奶油黄为主）；章节页/封面可用高饱和纯色（如 #06b6d4 青色）制造波普反差页。
- 粗体排版和大字号是视觉主体；文字可倾斜、错位或重叠。
- 可用 CSS 绘制放射线、星星、圆点、半调网点等复古印刷图案。
- 雕塑方言：白色圆形/椭圆 CSS 形状模拟古典雕塑轮廓，叠加现代物件符号（🕶 🎧 📱），颜色与背景形成强对比；不重复用同一形状作所有页面主角。

## 配色（CSS 可用）
- 背景：#fef3c7（奶油黄）；反差页 #06b6d4（青色）
- 文字：#451a03（深棕）；反差页 #ffffff（白色）
- 强调：#dc2626（红色）和 #f59e0b（橙色）
- 辅助：#92400e（中棕）
- 面板：#fffbeb（浅黄）

## 排版（HTML/CSS）
- 标题用超粗无衬线（font-weight: 900; font-size: 36-52px），可倾斜（skewX(-5deg) 或 rotate(-2deg)）。
- 正文用粗体无衬线（font-weight: 700; font-size: 14-16px），短句化。
- 标题可与主体局部重叠，制造杂志封面冲击力。

## 推荐版式（960×540pt）
- 海报封面：超大标题 + 放射线背景 + 底部信息条。
- 对比页：左右分栏，不同复古色块表达 Old/New 或 A/B。
- 数据页：巨大数字 + 复古色条。
- 信息页：主体形状 + 旁边气泡（不对称 border-radius）标注要点。

## CSS 实现要点
- 放射线：`background: repeating-conic-gradient(#f59e0b 0deg 10deg, transparent 10deg 20deg);`
- 圆点：`background-image: radial-gradient(#dc2626 2px, transparent 2px); background-size: 16px 16px;`
- 复古边框：`border: 4px solid #451a03;`；倾斜文字：`transform: skewX(-5deg);`
- 雕塑质感：`border-radius: 50%` + `box-shadow: inset -10px -10px 20px rgba(0,0,0,0.1)`。

## 避免
- 不使用现代极简风格或低对比配色。
- 不使用过多颜色，保持复古色调统一；不让装饰物覆盖核心文字。
- 不让文字过小，破坏海报感。

## 输出要求
保留输入内容的信息结构和文字含义，按此风格重绘为统一连贯的 16:9 演示页面。所有视觉元素必须用 HTML/CSS 实现，不依赖外部图片资源。
