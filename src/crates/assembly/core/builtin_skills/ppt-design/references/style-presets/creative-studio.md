# 黑橙创意（Black-Orange Studio）

## 风格标识
`creative-studio`

## 适用场景
创意机构提案、品牌策略、产品发布、视觉案例、营销方案等需要干练时尚感的商业演示。

## 风格定位
Creative Studio。白底、黑字、血橙强调色构成清爽锐利的创意机构风格，搭配动态排版与少量英文版式标签。

## 视觉系统
- 背景以白色为主（#ffffff），保持干净和高对比。
- 文字以黑色为主，血橙色用于重点、线条、编号和交互感元素。
- 可用 CSS 抽象动态形状（倾斜矩形、斜切块）模拟运动感；照片区域用渐变或纯色块替代，配合硬裁切、斜切。
- 少量英文标签可作版式元素，但正文遵循用户指定语言。

## 配色（CSS 可用）
- 背景：#ffffff（白色）
- 文字：#171717（近黑）
- 强调：#ea580c（血橙）
- 辅助：#525252（中灰）
- 面板：#fafafa（极浅灰）

## 排版（HTML/CSS）
- 标题用大胆无衬线（font-weight: 800; font-size: 32-44px）。
- 正文用短段落、编号列表或清晰卡片。
- 大编号（font-size: 64-80px; color: #ea580c; opacity: 0.3）可作背景装饰；侧边竖排小字（writing-mode: vertical-rl）增强机构感。

## 推荐版式（960×540pt）
- 白底大标题 + 血橙编号 + 斜切形状（clip-path: polygon(...)）。
- 左侧策略文字，右侧动态色块裁切。
- 案例页：色块网格 + 橙色标签。
- 结论页：黑色大字 + 橙色下划线（border-bottom: 3px solid #ea580c）强调一句核心判断。

## CSS 实现要点
- 斜切：`clip-path: polygon(0 0, 100% 10%, 100% 100%, 0 90%)`。
- 背景大编号：`position: absolute; z-index: 0; opacity: 0.15;`
- 下划线用 `border-bottom: 4px solid #ea580c` 而非 text-decoration。
- 卡片左侧强调：`border-left: 4px solid #ea580c`。

## 避免
- 不使用黑橙满屏高压配色；橙色只作强调。
- 不使用花哨渐变、卡通图标或复杂纹理。
- 不让英文装饰替代用户指定语言的核心内容。

## 输出要求
保留输入内容的信息结构和文字含义，按此风格重绘为统一连贯的 16:9 演示页面。所有视觉元素必须用 HTML/CSS 实现，不依赖外部图片资源。
