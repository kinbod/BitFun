# 暗黑霓虹（Dark Neon）

## 风格标识
`dark-neon`

## 适用场景
科技发布、AI/系统架构、前沿研究、数据分析、赛博朋克/前卫艺术主题等需要未来感与视觉冲击的演示。

## 风格定位
Dark Neon。深色背景加霓虹强调，提供两种方言：故障艺术（叛逆、数字破坏感）与霓虹制图（理性、建筑制图感）。根据主题选定一种方言后全 deck 保持一致。

## 视觉系统
- 背景使用纯黑（#0a0a0a）或深暖灰（#1c1917）。
- 故障方言：CSS 模拟故障效果——文字错位、青/品红颜色分离、扫描线、噪点；霓虹青与品红作强调。
- 制图方言：叠加极细网格线背景，CSS 线框图、技术示意图、几何拼贴；霓虹黄几何元素作核心强调，配 Fig. 01 式等宽字体标签与细指引线。
- 图层顺序：网格/暗背景 → 霓虹几何 → 主体内容 → 前景文字标注。

## 配色（CSS 可用）
- 背景：#0a0a0a（纯黑）或 #1c1917（深暖灰）
- 文字：#e5e5e5 / #fafaf9（浅灰白）
- 强调（故障）：#22d3ee（青色）和 #f472b6（品红）
- 强调（制图）：#facc15（霓虹黄）
- 辅助：#737373 / #a8a29e（中灰）
- 面板：#171717 / #292524（深灰）

## 排版（HTML/CSS）
- 标题用粗体无衬线（font-weight: 900; font-size: 32-48px; letter-spacing: 0.05em），可带霓虹光晕。
- 正文小字号（14-16px）严格对齐；编号、图注、坐标标签用等宽字体（monospace）增加科技感。

## 推荐版式（960×540pt）
- 故障标题：大标题带颜色分离（多层 text-shadow）。
- 霓虹数据：巨大数字 + 霓虹色 + 扫描线背景。
- 技术制图：线框图居中（多层 div 边框嵌套），周围细线标注模块。
- 几何流程：点线/虚线（border-style: dashed）连接霓虹几何表达逻辑流。

## CSS 实现要点
- 故障：`text-shadow: 2px 0 #22d3ee, -2px 0 #f472b6;`；光晕：`text-shadow: 0 0 10px #22d3ee, 0 0 20px #22d3ee;`
- 扫描线：`background: repeating-linear-gradient(0deg, transparent, transparent 2px, rgba(0,0,0,0.3) 2px, rgba(0,0,0,0.3) 4px);`
- 网格背景：`background-image: linear-gradient(rgba(255,255,255,0.05) 1px, transparent 1px); background-size: 40px 40px;`
- 霓虹线条：`border: 1px solid #facc15` + `text-shadow: 0 0 10px rgba(250,204,21,0.3)`。

## 避免
- 不使用明亮背景或传统商务配色，破坏暗黑氛围。
- 不让故障效果过度导致文字不可读；不让制图线和标注压过核心内容。
- 不混用两种方言的强调色系；一套 deck 内保持统一。

## 输出要求
保留输入内容的信息结构和文字含义，按此风格重绘为统一连贯的 16:9 演示页面。所有视觉元素必须用 HTML/CSS 实现，不依赖外部图片资源。
